                                  twidemaru.mac
                        秀丸(で編集)の ついでに、twitter。
                             秀丸 Ver.8 (以上) 専用版

          秀丸で作業している合間でも、ふとつぶやけたら便利かな、と。
          今書いているその内容を、それに関した内容を、さくっとその場でtweetに。
          つぶやく以外はできない簡単マクロ。

1. 説明
1-1. 概要。これはなに？
 　秀丸エディッタ用のマクロです。
 　twitterというマイクロブログに、現在の編集内容をつぶやくためのマクロです。

1-2. だれとく？
 　秀丸を使って作業・編集をされることが多く、かつそれに関すること(進捗・備忘・感想etc)をtwitterに書き込んでいるような場合にお役に立てると思います。
 　そのような環境で、たとえば現在編集している文章をそのままor一部を改変してつぶやくことが多いとか、そもそも投稿するために専用アプリやブラウザに切り替えるのが面倒くさいとお考えであればお試しください。
 　もちろん、新規ウィンドウを立ち上げて脈絡なくつぶやくというような使い方もできます。
 　なお、自称初心者の方の使用はお勧めしていません。

1-3. で、具体的な機能は？
 　起動すると、現在編集中の論理行全体を対象にして、その文字列を使ってtwitterにつぶやきます。
 　(矩形でない)範囲選択をしていた場合は、対象をその選択範囲にします。
 　

1-4. それ以外の機能は？
 　範囲選択されていない場合は、新しいウィンドウを立ち上げます。
 　そのウインドウでもう一度マクロを実行したら、そのウィンドウの内容を投稿します。

1-5. できそうでできない機能は何？
 　今のところ入力専用で、表示できません。従って、Retweetのようなこともできません。

4. 必要なもの

 　以下のような環境が必要です。他にもあるかもしれませんが、一般的なユーザなら多分大丈夫でしょう。

 ・秀丸エディッタ Ver 8以上が動作する。
 ・COM・秀丸エディッタ同梱のDLLなどの呼び出し・実行できる権限がある。
 ・マクロ本体が存在するフォルダ及びそこに存在するファイルに対して、読み取り・実行・書き込みなどができる権限がある。

 ・マクロ内で設定した場所(デフォルトではマクロと同じフォルダ)に "フォローしている人の一覧" を記したファイルを作成・読み取ることができる。
